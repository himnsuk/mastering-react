 Congratulations on completing the final application in this course! With MemeGenerator completed, you will always have a nice reference for how to use middleware and handle asynchronous actions with redux.

 Now, you can celebrate! You’ve made it, and can create complex web applications using redux.

 ***

 Here’s a brief overview of the main concepts that appeared in this section

  • Asynchronous actions complete at an unforeseeable amount of time. But redux allows us to have action creators that return functions to handle asynchronous actions through middleware.

  • Middleware lies between the request and response layer of redux and runs some code, i.e. debugging and logging.

  • The x-www-form-encoded represents a type of application content that certain apis allow their POST endpoints to accept data from.
